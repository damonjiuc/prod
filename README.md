```markdown
# Сайт для холдинга

## Направления
- **Розница**
- **Опт**
- **Партнерская программа**

Для партнерской программы предусмотрен личный кабинет, в который подтягиваются данные из 1С из xls файлов, обработчик `/app/main.py`.

## Структура проекта

### /app
Папка приложения:
- `__init__.py` - запуск приложения
- `config.py` - задаются параметры для авторизации в БД и прочее, переменные задаются в `.env` в корне
- `extensions.py` - для подключения библиотек
- `main.py` - скрипт для обновления заказов дизайнеров

### /models
Классы для привязки приложения к БД MySQL

- **/models/items.py**
Таблица для предметов из заказов дизайнеров

- **/models/orders.py**
Таблица для заказов дизайнеров

- **/models/users.py**
Таблица для пользователей

### /routes
Роуты, в которых прописаны обработчики и вывод страниц по категориям

- **/routes/admin.py**
Админ часть

- **/routes/main.py**
Индекс и статика

### /static
Папка для размещения CSS, JS, картинок и шрифтов

- **/static/css**
Стили веб-приложения:
- `reset.css` - сбрасывает стандартные стили браузера
- `fonts.css` - для подключения шрифтов
- `var.css` - для объявления переменных
- `style.css` - для основных стилей сайта, которые всегда подгружаются

- **/static/css/blocks/admin.css**
Стили, нужные только в админке

### /templates
Файлы шаблона сайта на Jinja2, разделены по категориям страниц

- **/templates/main**
Основные шаблоны сайта (индекс, статика, хэдер, футер)

- **/templates/admin**
Шаблоны админ части

### /xls
Эксель файлы, выгруженные из 1С
```
